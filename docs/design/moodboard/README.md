# Moodboard

วางไฟล์ PNG ที่ export จาก Figma ไว้ในโฟลเดอร์นี้

ยิ่งครอบคลุมยิ่งดี — สิ่งที่อยากเห็น:
- หน้าแรก / hero
- การ์ดฟอนต์
- ปุ่ม (ทุก variant ถ้ามี)
- ตัวอย่างตัวอักษร (heading + body)
- palette / swatch สี

ไฟล์พวกนี้ commit เข้า git ด้วย — ทิศทางดีไซน์ที่ไม่มีภาพอ้างอิงจะเน่าภายใน 2 session
